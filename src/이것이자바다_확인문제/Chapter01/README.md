## 01. 자바 언어의 특징

### Q. 자바 언어의 특징을 잘못 설명한 것은 무엇입니까?

- [ ] ➊ 안드로이드 애플리케이션뿐만 아니라 웹 사이트를 개발할 때 사용하는 언어이다.
- [ ] ➋ 한 번 작성으로 다양한 운영체제에서 실행할 수 있다.
- [ ] ➌ 객체 지향 프로그래밍 언어이다.
- [x] ➍ 개발자가 코드로 메모리를 관리해야 한다.

> **정답 : ➍**
>
> **해설 :**  
> 자바는 **Garbage Collector(GC)** 가 자동으로 메모리를 관리한다.  
> 따라서 개발자가 직접 메모리를 해제하지 않아도 된다.

## 02. JDK의 종류 (OpenJDK vs Oracle JDK)

### Q. OpenJDK와 Oracle JDK를 잘못 설명한 것은 무엇입니까?

- [ ] ➊ 둘 다 학습용 및 개발용으로 무료 사용 가능하다.
- [x] ➋ Oracle JDK는 개발 소스 공개 의무가 없지만, OpenJDK는 있다.
- [ ] ➌ 둘 다 Java SE 구현체이다.
- [ ] ➍ JDK 11 LTS 이후의 다음 LTS 버전은 JDK 17이다.

> **정답 : ➋**
>
> **해설 :**  
> OpenJDK는 오픈소스이며 GPL 라이선스를 따른다.  
> Oracle JDK는 상업적 사용 시 라이선스 정책을 확인해야 한다.

## 03. 환경 변수

### Q. 환경 변수에 대해 잘못 설명한 것은 무엇입니까?

- [ ] ➊ 운영체제가 프로그램 실행을 위해 관리한다.
- [ ] ➋ JAVA_HOME은 JDK 설치 경로를 가진다.
- [ ] ➌ PATH는 명령어 파일을 찾을 때 사용된다.
- [x] ➍ 환경 변수 수정 시 기존 터미널에 바로 적용된다.

> **정답 : ➍**
>
> **해설 :**  
> 환경 변수를 변경해도 기존 터미널에는 즉시 반영되지 않는다.  
> 터미널을 새로 실행해야 적용된다.

## 04. 자바 가상 머신 (JVM)

### Q. JVM에 대해 잘못 설명한 것은 무엇입니까?

- [ ] ➊ JVM은 java.exe 명령어로 실행된다.
- [ ] ➋ JVM은 바이트코드를 기계어로 변환한다.
- [x] ➌ JVM은 운영체제에 독립적이다.
- [ ] ➍ 바이트코드는 어떤 JVM에서도 실행 가능하다.

> **정답 : ➌**
>
> **해설 :**  
> 자바 프로그램은 운영체제에 독립적이지만,  
> JVM 자체는 운영체제별로 다르게 구현된다.

## 05. 자바 프로그램 개발 과정

### Q. 자바 프로그램 개발 과정을 순서대로 작성하세요.

### 정답
`( ➌ ) → ( ➊ ) → ( ➋ ) → ( ➍ )`

- [ ] ➊ javac.exe로 바이트코드 파일(.class)을 생성한다.
- [ ] ➋ java.exe로 JVM을 실행한다.
- [ ] ➌ 자바 소스 파일(.java)을 작성한다.
- [ ] ➍ JVM이 main() 메소드를 실행한다.

> **해설 :**  
> 자바는 `.java` 소스를 작성한 뒤 컴파일하여 `.class` 파일을 생성하고,  
> JVM이 이를 실행하는 구조이다.

## 06. 자바 소스 파일 작성 규칙

### Q. 자바 소스 파일 작성 규칙 중 잘못된 것은 무엇입니까?

- [ ] ➊ 파일명과 클래스명은 동일해야 한다.
- [ ] ➋ 클래스와 메소드 블록은 `{ }` 로 감싼다.
- [ ] ➌ 실행문 끝에는 세미콜론(;)을 붙인다.
- [x] ➍ 주석은 문자열 안에도 작성할 수 있다.

> **정답 : ➍**
>
> **해설 :**  
> 문자열 내부의 `//`, `/* */` 는 주석이 아니라 문자열로 인식된다.

## 07. Eclipse IDE

### Q. Eclipse의 특징으로 올바른 것을 모두 선택하세요.

- [x] ➊ 오픈소스 IDE이다.
- [x] ➋ 저장 시 자동 컴파일된다.
- [x] ➌ Workspace는 프로젝트 기본 디렉토리이다.
- [x] ➍ Java 17 지원 최소 버전은 Eclipse 2021-12이다.

> **정답 : ➊ ➋ ➌ ➍**
>
> **해설 :**  
> Eclipse는 대표적인 오픈소스 IDE이며,  
> 저장 시 자동 컴파일 기능(Build Automatically)을 제공한다.  
> 또한 Workspace는 프로젝트가 저장되는 기본 작업 공간을 의미한다.
