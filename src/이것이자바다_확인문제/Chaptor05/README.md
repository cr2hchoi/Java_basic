# 📝 이것이 자바다 확인문제 ch05

## 1. 참조 타입의 특징
**Q. 참조 타입에 대한 설명으로 틀린 것은 무엇입니까?**
- [ ] ➊ 참조 타입에는 배열, 열거, 클래스, 인터페이스가 있다.
- [ ] ➋ 참조 타입 변수의 메모리 생성 위치는 스택이다.
- [ ] ➌ 참조 타입에서 ==, != 연산자는 객체 번지를 비교한다.
- [x] ➍ 참조 타입은 null 값으로 초기화할 수 없다.

> **정답: ➍**
> **해설**: 참조 타입 변수는 객체를 참조하지 않는다는 의미로 `null` 값을 가질 수 있다. `null`로 초기화된 변수는 스택 영역에 생성되며, 어떤 객체도 참조하지 않는 상태를 나타낸다.

## 2. 자바의 메모리 사용
**Q. 자바에서 메모리 사용에 대한 설명으로 틀린 것은 무엇입니까?**
- [ ] ➊ 로컬 변수는 스택 영역에 생성되며 실행 블록이 끝나면 소멸된다.
- [ ] ➋ 메소드 코드나 상수, 열거 상수는 정적(메소드) 영역에 생성된다.
- [x] ➌ 참조되지 않는 객체는 프로그램에서 직접 소멸 코드를 작성하는 것이 좋다.
- [ ] ➍ 배열 및 객체는 힙 영역에 생성된다.

> **정답: ➌**
> **해설**: 자바는 가비지 컬렉터(GC)가 참조되지 않는 객체를 자동으로 찾아 메모리에서 제거하므로, 프로그래머가 직접 객체를 소멸시키는 코드를 작성할 필요가 없으며 작성할 수도 없다.

## 3. String 타입의 특징
**Q. String 타입에 대한 설명으로 틀린 것은 무엇입니까?**
- [ ] ➊ String은 클래스이므로 참조 타입이다.
- [x] ➋ String 타입의 문자열 비교는 ==를 사용해야 한다.
- [ ] ➌ 동일한 문자열 리터럴을 저장하는 변수는 동일한 String 객체를 참조한다.
- [ ] ➍ new String (“문자열”)은 문자열이 동일하더라도 다른 String 객체를 생성한다.

> **정답: ➋**
> **해설**: 참조 타입인 String 객체의 내부 문자열 값을 비교할 때는 `==` 연산자가 아닌 `equals()` 메소드를 사용해야 한다. `==` 연산자는 객체의 주소값을 비교하기 때문에 문자열 내용이 같아도 `false`가 나올 수 있다.

## 4. 배열 생성
**Q. 배열을 생성하는 방법으로 틀린 것은 무엇입니까?**
- [ ] ➊ `int[] array = { 1, 2, 3 };`
- [x] ➋ `int[] array; array = { 1, 2, 3 };`
- [ ] ➌ `int[] array = new int[3];`
- [ ] ➍ `int[][] array = new int[3][2];`

> **정답: ➋**
> **해설**: 배열 변수를 먼저 선언한 후 값 목록으로 배열을 생성할 때는 `new int[] { 1, 2, 3 };`과 같이 `new` 키워드를 반드시 사용해야 한다. 중괄호 `{ }`만 사용하는 초기화 방식은 배열 변수 선언과 동시에 할 때만 가능하다.

## 5. 배열의 기본 초기값
**Q. 배열의 기본 초기값에 대한 설명으로 틀린 것은 무엇입니까?**
- [ ] ➊ 정수 타입 배열 항목의 기본 초기값은 0이다.
- [ ] ➋ 실수 타입 배열 항목의 기본 초기값은 0.0f 또는 0.0이다.
- [x] ➌ boolean 타입 배열 항목의 기본 초기값은 true이다.
- [ ] ➍ 참조 타입 배열 항목의 기본 초기값은 null이다.

> **정답: ➌**
> **해설**: `boolean` 타입 배열 항목의 기본 초기값은 `true`가 아니라 `false`이다. 자바에서 배열을 생성할 때 별도의 값을 지정하지 않으면 각 타입의 기본값으로 자동 초기화된다.

## 6. 배열 길이 출력

## 7. 배열의 최대값 구하기
**Q. 주어진 배열 항목에서 최대값을 출력하는 코드를 작성해보세요(for 문 이용).**
> int[] array = { 1, 5, 3, 8, 2 };

```java
int[] array = { 1, 5, 3, 8, 2 };
int max = 0;
for (int i = 0; i < array.length; i++) {
    if (max < array[i]) {
        max = array[i];
    }
}
System.out.println("최대값: " + max);
```

* 해설: 배열의 첫 번째 요소부터 마지막 요소까지 순차적으로 비교하며, 현재까지의 최대값(max)보다 큰 값이 나타나면 max를 해당 값으로 갱신하는 방식을 사용한다.

## 8. 배열의 합과 평균 구하기
**Q. 주어진 배열 항목의 전체 합과 평균을 구해 출력하는 코드를 작성해보세요(중첩 for 문 이용).**

```java
int[][] array = {
    {95, 86},
    {83, 92, 96},
    {78, 83, 93, 87, 88}
};
```
* 정답 
```

int sum = 0;
int count = 0;

for (int i = 0; i < array.length; i++) {
    for (int j = 0; j < array[i].length; j++) {
        sum += array[i][j];
        count++;
    }
}

double avg = (double) sum / count;

System.out.println("전체 합: " + sum);
System.out.println("평균: " + avg);
```
* 해설
  * 바깥쪽 for 문은 행을 순회하고, 안쪽 for 문은 각 행에 포함된 열(항목)을 순회한다. 전체 합을 구할 때는 sum 변수를 사용하며, 평균을 구할 때는 총 항목 개수(count)를 나누어 계산한다. 나눗셈 시 실수 결과를 얻기 위해 형변환((double))을 수행한다.

## 9. 학생 점수 분석 프로그램

* 정답 
```java
import java.util.Scanner;

public class StudentScoreAnalysis {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        int[] scores = null;
        int studentNum = 0;
        boolean run = true;

        while (run) {
            System.out.println("------------------------------------------------------");
            System.out.println("1.학생수 | 2.점수입력 | 3.점수리스트 | 4.분석 | 5.종료");
            System.out.println("------------------------------------------------------");
            System.out.print("선택> ");
            
            int selectNo = Integer.parseInt(scanner.nextLine());

            if (selectNo == 1) {
                System.out.print("학생수> ");
                studentNum = Integer.parseInt(scanner.nextLine());
                scores = new int[studentNum];
            } else if (selectNo == 2) {
                for (int i = 0; i < scores.length; i++) {
                    System.out.print("scores[" + i + "]> ");
                    scores[i] = Integer.parseInt(scanner.nextLine());
                }
            } else if (selectNo == 3) {
                for (int i = 0; i < scores.length; i++) {
                    System.out.println("scores[" + i + "]: " + scores[i]);
                }
            } else if (selectNo == 4) {
                int max = 0;
                int sum = 0;
                for (int score : scores) {
                    if (score > max) max = score;
                    sum += score;
                }
                System.out.println("최고 점수: " + max);
                System.out.println("평균 점수: " + (double) sum / studentNum);
            } else if (selectNo == 5) {
                run = false;
            }
        }
        System.out.println("프로그램 종료");
    }
}
``` 
* 해설


1) 학생 수(1번)를 입력받아 배열 scores의 크기를 결정한다.

2) 점수 입력(2번)을 통해 배열에 값을 채운다.

3) 분석(4번) 시 for 문을 사용해 max 변수와 sum 변수를 갱신하여 결과값을 산출한다.

4) scanner.nextLine()을 사용하면 엔터키 처리 문제 없이 정수형 입력을 안전하게 변환할 수 있다.
