## 01. 참조 타입의 특징

### Q. 참조 타입에 대한 설명으로 틀린 것은 무엇입니까?

- [ ] ➊ 참조 타입에는 배열, 열거, 클래스, 인터페이스가 있다.
- [ ] ➋ 참조 타입 변수의 메모리 생성 위치는 스택이다.
- [ ] ➌ 참조 타입에서 `==`, `!=` 연산자는 객체 번지를 비교한다.
- [x] ➍ 참조 타입은 `null` 값으로 초기화할 수 없다.

> **정답 : ➍**
>
> **해설 :**  
> 참조 타입 변수는 객체를 참조하지 않는 상태를 의미하는 `null` 값을 가질 수 있다.
>
> `null`로 초기화된 변수는 어떤 객체도 참조하지 않는 상태를 나타낸다.

## 02. 자바의 메모리 사용

### Q. 자바 메모리 사용에 대한 설명으로 틀린 것은 무엇입니까?

- [ ] ➊ 로컬 변수는 스택 영역에 생성되며 실행 블록이 끝나면 소멸된다.
- [ ] ➋ 메소드 코드나 상수, 열거 상수는 메소드 영역에 생성된다.
- [x] ➌ 참조되지 않는 객체는 프로그램에서 직접 소멸 코드를 작성하는 것이 좋다.
- [ ] ➍ 배열 및 객체는 힙 영역에 생성된다.

> **정답 : ➌**
>
> **해설 :**  
> 자바에서는 가비지 컬렉터(GC)가 사용되지 않는 객체를 자동으로 제거한다.
>
> 따라서 개발자가 직접 객체를 소멸시키는 코드를 작성하지 않는다.

## 03. String 타입의 특징

### Q. String 타입에 대한 설명으로 틀린 것은 무엇입니까?

- [ ] ➊ String은 클래스이므로 참조 타입이다.
- [x] ➋ String 타입의 문자열 비교는 `==`를 사용해야 한다.
- [ ] ➌ 동일한 문자열 리터럴은 같은 String 객체를 참조한다.
- [ ] ➍ `new String("문자열")`은 동일한 문자열이어도 새로운 객체를 생성한다.

> **정답 : ➋**
>
> **해설 :**  
> String 객체의 문자열 내용을 비교할 때는 `equals()` 메서드를 사용해야 한다.
>
> `==` 연산자는 문자열 내용이 아니라 객체의 주소값을 비교한다.

## 06. 배열 길이 출력

### Q. 다음 코드의 실행 결과를 작성하세요.

```java
int[][] array = {
    {95, 86},
    {83, 92, 96},
    {78, 83, 93, 87, 88}
};

System.out.println(array.length);
```

>
> ```text
> 3

> ```
>

## 07. 배열의 최대값 구하기

### Q. 주어진 배열 항목에서 최대값을 출력하는 코드를 작성하세요.

```java
int[] array = {1, 5, 3, 8, 2};

int max = 0;

for (int i = 0; i < array.length; i++) {
    if (max < array[i]) {
        max = array[i];
    }
}

System.out.println("최대값: " + max);
```

> **해설 :**  
> 배열의 각 요소를 순차적으로 비교하면서 현재 최대값보다 큰 값이 나오면  
> `max` 변수 값을 갱신한다.

## 08. 배열의 합과 평균 구하기

### Q. 중첩 for 문을 이용하여 배열 항목의 전체 합과 평균을 구하는 코드를 작성하세요.

```java
int[][] array = {
    {95, 86},
    {83, 92, 96},
    {78, 83, 93, 87, 88}
};
```

### 정답

```java
int sum = 0;
int count = 0;

for (int i = 0; i < array.length; i++) {
    for (int j = 0; j < array[i].length; j++) {
        sum += array[i][j];
        count++;
    }
}

double avg = (double) sum / count;

System.out.println("전체 합: " + sum);
System.out.println("평균: " + avg);
```

> **해설 :**  
> 바깥쪽 반복문은 행(row)을 순회하고,  
> 안쪽 반복문은 각 행의 열(column)을 순회한다.
>
> 전체 합은 `sum`에 누적하고,  
> 평균은 전체 합을 항목 개수(`count`)로 나누어 계산한다.

## 09. 학생 점수 분석 프로그램

### Q. 학생 수와 점수를 입력받아 최고 점수와 평균 점수를 출력하는 프로그램을 작성하세요.

### 실행 예시

```text
------------------------------------------------------
1.학생수 | 2.점수입력 | 3.점수리스트 | 4.분석 | 5.종료
------------------------------------------------------
선택> 1
학생수> 3

------------------------------------------------------
1.학생수 | 2.점수입력 | 3.점수리스트 | 4.분석 | 5.종료
------------------------------------------------------
선택> 2
scores[0]> 85
scores[1]> 95
scores[2]> 93

------------------------------------------------------
1.학생수 | 2.점수입력 | 3.점수리스트 | 4.분석 | 5.종료
------------------------------------------------------
선택> 3
scores[0]: 85
scores[1]: 95
scores[2]: 93

------------------------------------------------------
1.학생수 | 2.점수입력 | 3.점수리스트 | 4.분석 | 5.종료
------------------------------------------------------

최고 점수: 95
평균 점수: 91.0

------------------------------------------------------
1.학생수 | 2.점수입력 | 3.점수리스트 | 4.분석 | 5.종료
------------------------------------------------------

프로그램 종료
```

### 정답

```java
import java.util.Scanner;

public class StudentScoreAnalysis {

    public static void main(String[] args) {

        Scanner scanner = new Scanner(System.in);

        int[] scores = null;
        int studentNum = 0;
        boolean run = true;

        while (run) {

            System.out.println("------------------------------------------------------");
            System.out.println("1.학생수 | 2.점수입력 | 3.점수리스트 | 4.분석 | 5.종료");
            System.out.println("------------------------------------------------------");
            System.out.print("선택> ");

            int selectNo = Integer.parseInt(scanner.nextLine());

            if (selectNo == 1) {

                System.out.print("학생수> ");
                studentNum = Integer.parseInt(scanner.nextLine());

                scores = new int[studentNum];

            } else if (selectNo == 2) {

                for (int i = 0; i < scores.length; i++) {
                    System.out.print("scores[" + i + "]> ");
                    scores[i] = Integer.parseInt(scanner.nextLine());
                }

            } else if (selectNo == 3) {

                for (int i = 0; i < scores.length; i++) {
                    System.out.println("scores[" + i + "]: " + scores[i]);
                }

            } else if (selectNo == 4) {

                int max = 0;
                int sum = 0;

                for (int score : scores) {
                    if (score > max) {
                        max = score;
                    }

                    sum += score;
                }

                System.out.println("최고 점수: " + max);
                System.out.println("평균 점수: " + (double) sum / studentNum);

            } else if (selectNo == 5) {

                run = false;
            }
        }

        System.out.println("프로그램 종료");
    }
}
```

> **해설 :**
>
> 1. 학생 수 입력 시 배열의 크기를 결정한다.
> 2. 점수 입력 기능으로 배열에 데이터를 저장한다.
> 3. 분석 기능에서는 최고 점수와 총합을 계산한다.
> 4. `scanner.nextLine()`을 사용하면 입력 버퍼 문제를 줄일 수 있다.
